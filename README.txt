Billing TEAM - as part of fullstack developers course in Shenkar collage

Project Description -
    this project consist of several fetchers:
        1) CRUD costumer feedback
        2) Live chat with support
        3) feedback data saved automatically in google-sheets
        4) the data saved in json file
        5) API DOCS to all of those fetchers
        6) UI design for user interface

How to run this project -
    1) clone git repository : 'https://github.com/Galor-sw/Billing-TEAM.git'
    2) add file: 'credentials.json'
    3) copy to credentials.json the project password's - ask the TEAM LEADER for it
    4) run: 'npm run start'
    5) run: 'http://localhost:8080/'


How to use the project:
    1) insert one of existing email from 'feedback.json'
    2) fill your feedback
    3) send the feedback to save it in our database
    4) you can press 'help' to contact our support in live chat
    5) login again with your mail so you can see your feedback and edit it

what we installed:
    1) npm init -f
    2) npm i nodemon --save--dev
    3) npm i winston
    4) npm i google-spreadsheet --save
    5) npm i socket.io
    6) npm i moment
    7) npm i --save--dev eslint
